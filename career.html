Senior Web Server(Linux) Administrator

The successful Server Administrator will be responsible for:
  • Setup new server environments for websites, apps or other applications
  • Deploy websites, apps or other applications
  • Monitor servers and fix server failures in the shortest time
  • Mange users and security
  • Write scripts for server maintenance automation
To be considered for this exciting career-growth opportunity, you must possess the following skills/experience:
  • Solid knowledge of HTTP (1.0, 1.1 ), DNS and CDN
  • Expertised in server-script coding (Python, Node.js, Perl, Shell etc)
  • 5 years minimum experience as an Administrator in a LNMP/LAMP environment
  • Solid skills in Linux administration
  • Solid skills in nginx/apache/php configaration and trouble shooting
  • Experienced in deploying and managing server monitoring solutions
  • Familiar with cloud services such AWS
